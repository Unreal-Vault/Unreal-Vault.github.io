Unreal Engine Asset Management Library


Your own personal asset cloud

Browse and Load your asset packs into any Unreal project

Configure everything through JSON

Vault settings are broken down into local user settings, and global cross-project settings, allowing any sized team to work efficently without having to re-set things up between projects

The local settings live with the plugin, and define the local users choices, as well as the global file locations

The global settings define the project-agnostic rules, such as the library location

The Tag pool is also saved as a JSON file, making it easy to modify by hand if required, such as during a re-organise, or if a bad tag was added

Metadata for all your packages is written out as .meta files, which is a json formatted document with all the information of the asset pack

Full Python & Blueprint Scripting Support

The UI handles everything you need, but if you wanted to automate or pipeline, The Vault also comes with a blueprint library, with full support for both Python and Editor Scripting through blueprint


Example Python for publishing:

import unreal as ue

selection = EditorUtilityLibrary.get_selected_assets()

ue.VaultLibrary.package(selection, selection[0].get_name(), ue.VaultLibrary.get_developer_name(), "AutoPackaged By Python")
